# GoCloud 

### 一个支持快速搭建支持私有化部署的个人私有云系统，又可以作为企业级分布式云存储解决方案。

## 功能特点(迭代中...):
1. 依托于golang开发，支持各种平台部署
2. 前端分片上传、秒传文件
3. 下载支持分片、断点续传，轻松跑满带宽
4. 支持文件分享、快速转存，以及安全验证，轻松分享自己的资源
4. 后端存储OSS(阿里云)、KODO(七牛云)、Ceph(私有分布式/aws)等

## 使用docker运行

mac OS, Linux 在根目录下运行
```
$ ./start.sh
```

Windows 在根目录下运行
```
$ start.sh
```
